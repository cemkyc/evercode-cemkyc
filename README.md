# **Task Scheduler Project**
Проект для отборочного этапа стажировки. Реализует систему планирования периодических задач с инкапсулированным логированием.

Выполнен первый модуль

## **Структура проекта**
- `Config/`       — конфигурация приложения
- `Errors/`       — кастомные классы ошибок
- `Logger/`       — централизованная система логирования
- `Schedule/`     — логика планировщика задач
- `Tasks/`        — периодические задачи
- `Validators/`   — валидация входных параметров
- `Middleware/`   — middlewar/   
- `Routes/`       — REST API маршруты
- `Services/`     — работа с внешними API
- `Storage/`
  
## **Как запустить**
1. `npm start` (В консоль, необходимо находиться в папке проекта)
2. `npm test` Для unit-тестов (В консоль, необходимо находиться в папке проекта)
