##Task Scheduler Project
Проект для отборочного этапа стажировки. Реализует систему планирования периодических задач с инкапсулированным логированием.

##Структура проекта
- `config.js` — Централизованные настройки (имя приложения).
- `logger.js` — Модуль вывода сообщений в консоль с форматированием.
- `scheduler.js` — Основной файл (точка входа). Содержит логику планировщика и запуск периодических задач.

##Как запустить
1. `npm start` (В консоль, необходимо находиться в папке проекта)
